Overtime and Leave Management System 
======

[Link](https://olms.shlib.cf)


Install
-------

Install OLMS

    $ pip install -e .

Run
---

    $ export FLASK_APP=OLMS
    $ export FLASK_ENV=development
    $ flask init-db
    $ flask run

Or on Windows cmd

    > set FLASK_APP=OLMS
    > set FLASK_ENV=development
    > flask init-db
    > flask run

Open http://127.0.0.1:5000 in a browser.
